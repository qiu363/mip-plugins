# mip-hk-showarticle

mip-hk-showarticle 显示文章更多详情

| 描述 | 显示文章更多详情|
|---|---|
|可用性	|完成 |
|所需脚本| mip-hk-showarticle |

# 使用方法

在MIP HTML中,直接使用标签, 用于显示文章更多详情。示例:
```
    <mip-hk-showarticle page='erji_detail_news'>
        <div id="J_show_art">
            <span>展开全文</span>
        </div>
    </mip-hk-showarticle>
```
# 属性

- **page**

    - 是否必填：否

    - 说明：统计参数
